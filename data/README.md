# Study data (not included in Git)

This repository ships **code only**. Participant data for the MexIHC pilot (*n* = 10) are published separately on Zenodo.

## After downloading the Zenodo archive

Extract into this folder so the layout matches:

```
data/
  Usuarios/U1/…          # EmotiBit CSV exports (EA, TH, PG, PR, PI, UN, …)
  self_report/cognitive/SUS.csv
  outputs/               # generated by pipeline/run_inference.py
  participants_registry/ # optional JSON manifests from the dashboard
```

Then restart the API. Charts read from `data/outputs/UX_activation_summary.csv` and `data/self_report/cognitive/SUS.csv`.
